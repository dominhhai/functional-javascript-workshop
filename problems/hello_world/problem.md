# Nhiệm vụ

## Tham số

* input: một chuỗi kí tự bất kì (String).

## Mẫu

```js
function upperCaser(input) {
  // TODO hoàn thiện đoạn mã này
}

// Đừng thay đổi dòng này
module.exports = upperCaser
```
